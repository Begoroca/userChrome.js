# Link Locationbar Lite
Beim Befahren eines Links wird dessen Adresse rechts in der Urlbar angezeigt, nachdem deren Icons dafür temporaer ausgeblendet 
werden.

Das **Ergebnis des Skripts**:

![Screenshot Link Locationbar Lite](https://github.com/ardiman/userChrome.js/raw/master/linkloactionbarlite/scr_linkloactionbarlite.png)

## Installation
Kopiere die uc.js-Datei in den Chromeordner des Profils.
